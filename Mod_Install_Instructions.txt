INSTALLATION:
1. Download the original game
2. Extract this mod
3. Copy the contents into the game's folder
	The game folder should be located in data\Facing Demons\Mods\Facing Demons
	Copy all the files from this mod folder into this location (you will need to press view game files in gamejolt)
4. Replace files when prompted
5. Read the ReadMe or ReadMe_TLDR (shorter version) for info about the mod
6. Launch the game

7. You can unistall the mod by unistalling the game and reinstalling it
